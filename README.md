# About

Implementation of [LMSR](https://mason.gmu.edu/~rhanson/mktscore.pdf) (Logarithmic Market Scoring Rules) automated market maker in Haskell. This engine is intended for use with artifical money, but aims to provide the same kind of guarantees that would enable you to use it to build real prediction markets. The more interesting thing this repo is trying to explore is to provide _information_ based on the markets outcomes.
